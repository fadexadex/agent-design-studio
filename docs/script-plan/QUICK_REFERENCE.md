# QUICK REFERENCE: Story-Driven Video Scripts

## 🚨 THE #1 RULE

**NEVER start with the brand name!**

The brand should appear ONLY ONCE, at the very end, after you've told the user's story.

---

## ✅ The Correct Story Structure

```
Act 1 (0-15%):  Problem/Desire  → NO BRAND
Act 2 (15-35%): Simple Solution → NO BRAND
Act 3 (35-60%): Magic Reveal    → NO BRAND
Act 4 (60-80%): Result/Outcome  → NO BRAND
Act 5 (80-100%): Value Summary  → BRAND APPEARS HERE!
```

---

## 📖 Act-by-Act Template

### Act 1: The Problem (NO BRAND)
**Text:** Ask a question or state a desire
- Example: "Got stuff you don't need?"
- Example: "Turn Books into Audio"
- Example: "Too many tasks?"

**Visual:** Simple object representing the problem
**Tone:** Calm, contemplative

---

### Act 2: The Solution (NO BRAND)
**Text:** Simple instruction + confirmation
- Pattern: "Just [action]" → "Done!"
- Example: "Just snap and post" → camera click → "Done!"
- Example: "Just drop and go" → upload → "Done!"

**Visual:** Phone/device, simple interaction, success indicator
**Tone:** Energetic, satisfying

---

### Act 3: The Magic (NO BRAND)
**Text:** Reveal the special capability
- Example: "Students on campus see it"
- Example: "Multiple Languages"
- Example: "Automatically prioritized"

**Visual:** Network connecting, multiplication, transformation
**Tone:** Exciting, surprising

---

### Act 4: The Result (NO BRAND)
**Text:** Show the successful outcome
- Example: "Meet. Exchange. Done."
- Example: "Distribute To Youtube"
- Example: "Done on time"

**Visual:** Transaction completing, parties connecting, success
**Tone:** Satisfying, accomplished

---

### Act 5: Brand Reveal (BRAND APPEARS!)
**Text:** 3-verb summary bouncing → Brand → Tagline
- Pattern: "[Verb 1]. [Verb 2]. [Verb 3]." → "BrandName" → "tagline"
- Example: "Sell. Find. Connect." → "Campor" → "Your campus economy"
- Example: "Translate. Dub. Distribute." → "LangEase"

**Visual:** Bouncing dot/star with text, morph into logo
**Tone:** Celebratory, conclusive

---

## 🎨 Text Choreography Patterns

### Catch-Up Pattern
```
"First word" enters → moves up
"Second word" enters delayed → catches up to align
Result: "First word second word" (aligned)
```

### Replacement Cycle
```
"Word 1" appears → holds → slides out left
"Word 2" enters from right → same position → slides out
"Word 3" replaces "Word 2"
```

### Bounce Sequence
```
Element bounces from top
Bounce 1: "Word 1" appears
Bounce 2: "Word 2" appears
Bounce 3: "Word 3" appears
```

### Makes Space Pattern
```
"Text" appears center
"Text" slides LEFT
Icon appears RIGHT (simultaneously)
```

## ❌ Common Mistakes

### Mistake 1: Brand Too Early
```
❌ Act 1: "Campor" appears
✅ Act 1: "Got stuff you don't need?"
```

### Mistake 2: No Story Arc
```
❌ Feature 1 → Feature 2 → Feature 3 → Brand
✅ Problem → Solution → Magic → Result → Brand
```

### Mistake 3: Brand Appears Multiple Times
```
❌ "Campor" in moments 1, 3, 5, 8
✅ "Campor" appears ONLY in final moment
```

### Mistake 4: Static Text
```
❌ "text": "Sell", "animation": "fade in"
✅ Full choreography: entrance (slide-left 600ms) + hold (2000ms) + exit (slide-left 400ms)
```

## ⚡ Quick Validation Checklist

Before finalizing a script, verify:

- [ ] Brand name does NOT appear in Acts 1-4
- [ ] Brand name appears ONLY ONCE in Act 5
- [ ] Act 1 establishes a problem or desire
- [ ] Act 2 shows a simple action → completion
- [ ] Act 3 reveals the "magic" capability
- [ ] Act 4 demonstrates the outcome
- [ ] Act 5 has 3 verbs bouncing → brand morph
- [ ] Every moment has text with full choreography
- [ ] Text has personality (playful/bold/calm/energetic)
- [ ] Story builds emotional investment

---

## 🎯 The Goal

Create videos where:
1. Viewers are emotionally invested in the story
2. The brand reveal feels earned and satisfying
3. Text performs choreographed movements
4. Story flows naturally from problem to brand

**Remember:** It's a story, not a business card!
